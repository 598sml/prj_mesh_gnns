# prj_mesh_gnns
Mesh GNNs (MeshGraphNet) for vortex shedding.

## Installation

### 1. Create and activate a virtual environment

```bash
python -m venv .venv
source .venv/bin/actiave
```

### 2. Install the requirements.txt

```bash
pip install -r requirements.txt
```

### 3. Install torch and torch specifics

```bash
pip install torch 
pip install torch-geometric
pip install torch-scatter torch-sparse -f https://data.pyg.org/whl/torch-2.11.0+cu130.html
```
