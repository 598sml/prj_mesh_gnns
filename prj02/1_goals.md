# Goals
The focus of this project is on implementing Graph Neural Network (MeshGraphNet) to predict vortex shedding for flow pass a cylinder. 

   - Short goal: understand GNNs architecture and start implementing the code.

   - Medium goal: implement the MeshGraphNet for the specified case and make inferences for different vortex shedding geometries.

   - Long goal: extend the problem to accomodate MeshGraphNet for immersed body problems, with structured eulerian meshes (fluid) and lagrangian meshes (solid).
